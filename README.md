# DACCS690C_HW2

Network analysis of the `hollywood.graphml` actor network using igraph and ggraph in R.

**Author:** Mark Scott

## Contents

- `Week_2_Homework.qmd` - source file
- `Week_2_Homework.html` - rendered report
- `Week_2_Homework.ipynb` - notebook version, runnable in Google Colab
- `hollywood.graphml` - network data

## What's inside

1. Node centrality measures (eigenvector, closeness, betweenness)
2. Correlation plot of the three measures, with interpretation
3. Network plot highlighting hub nodes
4. Communities investigation (transitivity test vs. random networks)
5. Community detection (Girvan-Newman and Louvain), with the best result plotted
